# AI Guides for TypeScript + Nest Repository

This directory contains narrow guide files for production-safe changes in a TypeScript repository with NestJS components, CLI entrypoints, local unit tests, and e2e tests.

Use the root `AGENTS.md` for always-on global rules.

Read additional files only when the task matches their scope:

- `./skills/typescript-style-and-typing-safety.md` — typing rules, style, DTO/config shape guidance
- `./skills/nest-and-cli-contract-safety.md` — Nest entrypoints, CLI behavior, public contracts
- `./skills/logger-and-helper-safety.md` — logger behavior and shared helper logic
- `./skills/test-placement-and-update-rules.md` — unit/e2e placement and narrow test updates
- `./skills/final-review-checklist.md` — final self-check before returning a patch
